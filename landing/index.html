FAQ

Common questions

What AI model does InterfaceForge use?

Pro subscribers use our hosted endpoint powered by GPT-4.1 Mini — no configuration needed. Free users can connect any OpenAI-compatible API (OpenAI, Anthropic via proxy, local models, etc.) by entering their own endpoint and key.

Is the generated code production-ready?

The exported code is clean, well-structured starter code with proper component architecture, design tokens, and responsive layout. It's designed to drop into an existing project and customize — not a pixel-perfect final product, but a major head start.

What if I don't have an API key?

InterfaceForge works without any API key. Without one, it generates using on-device template fallbacks — clearly labeled. Upgrade to Pro to use our hosted AI with no setup, or add your own key on Free.

Does it work on iPad and Mac?

Yes. InterfaceForge includes a dedicated iPad split-view layout with sidebar navigation. macOS support via Catalyst is coming soon — the iPad layout runs natively on Mac.

What export formats are supported?

React + CSS, Tailwind CSS, HTML + CSS, and SwiftUI. Each export includes the component code, a design tokens file, a README, and a LICENSE. More formats are planned.

Is my data private?

Prompts are sent only to your configured AI provider (or our proxy for Pro). Design history syncs via your private iCloud container — we never see it. The app collects no analytics or personal data.

Can I cancel Pro anytime?

Yes. Pro is a standard App Store subscription. Cancel anytime in your Apple ID settings. You keep Pro access until the end of your billing period, then revert to Free with no data loss.

How is this different from v0.dev / Bolt / Galileo?

Those are web-based tools for desktop workflows. InterfaceForge is the only native iOS app focused on mobile-first UI ideation. Snap a screenshot on your phone, generate a component, preview it, refine it, and export code — all without opening a browser.
