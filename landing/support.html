Support

Need help with InterfaceForge? Check the common issues below, or reach out directly.

Common Issues

🔑 "Add an API key first"

Open Settings (gear icon on Home screen) and paste your OpenAI API key. The key must start with sk-. Tap Test Connection to verify it works.

If you're a Pro subscriber, generation works without a key — make sure your subscription is active in Settings → Manage Subscription.

⚠️ Generation fails or returns an error

Check these things:

  1. Your API key is valid and has available credits
  2. The endpoint URL is correct (default: https://api.openai.com)
  3. You have an internet connection
  4. Try tapping Test Connection in Settings

If using a custom endpoint (local model, proxy), make sure it supports the OpenAI chat completions format.

📱 Template fallback instead of AI generation

If the app shows "Template fallback — AI unavailable", it means the AI endpoint couldn't be reached. The app generates a clearly labeled template result instead. To get AI results, verify your API key and connection in Settings.

🔄 iCloud sync not working

Make sure iCloud is enabled on your device: Settings → Apple ID → iCloud → Apps Using iCloud. InterfaceForge syncs via your private CloudKit container. Sync may take a few minutes after first launch.

💳 Subscription not recognized

If you purchased Pro but the app still shows Free limits:

  1. Open InterfaceForge → Settings → tap Restore Purchases
  2. Make sure you're signed in with the Apple ID used for the purchase
  3. Close and reopen the app

Apple subscription changes can take a few minutes to propagate.

📤 Export files won't open

Exported packages are shared as a folder via the iOS Share Sheet. If you're AirDropping to a Mac, the files arrive in your Downloads folder. Each export includes the component code, design tokens, a README, and a LICENSE file.

Managing Your Subscription

InterfaceForge Pro is an auto-renewable subscription managed through the App Store. To change or cancel:

  1. Open Settings on your iPhone or iPad
  2. Tap your name → Subscriptions
  3. Select InterfaceForge
  4. Change plan or tap Cancel Subscription

You keep Pro access until the end of your current billing period. No data is lost when downgrading to Free.

Feature Requests & Bug Reports

We'd love to hear from you. When reporting a bug, please include:

  • Your device model and iOS version
  • What you were doing when the issue occurred
  • The exact error message (if any)
  • Whether you're using BYOK or Pro hosted AI
