# App Store Screenshots Guide

## Required Sizes

### iPhone 6.7" (required)
- 1290 × 2796 px (iPhone 15 Pro Max / 16 Pro Max)
- Up to 10 screenshots

### iPhone 6.5" (required if supporting iPhone X and later)
- 1242 × 2688 px (iPhone 11 Pro Max)

### iPad 12.9" 6th gen (required for iPad)
- 2048 × 2732 px

## Recommended Screenshot Sequence (8 screens)

1. **Hero** — Home screen showing "Design polished UI from one clear prompt." with the Create with AI button
2. **Composer** — The prompt composer with a typed prompt, style chips, and template selector
3. **Generation** — AI generation in progress with progress bar and status messages
4. **Preview** — Generated component in live web preview with viewport controls
5. **Refinement** — Follow-up prompt bar with "Make it darker" chip, showing the refined result
6. **Export** — Export screen showing all 4 format options with file tree preview
7. **Gallery** — Community gallery with shared designs and fork button
8. **Pricing** — Paywall screen showing Free vs Pro comparison

## Screenshot Tips
- Use a clean device frame (Apple provides frames in the Marketing Resources)
- Dark theme (Aurora) shows best on the App Store dark background
- Include real generated content, not placeholder text
- Show the full iPhone notch / Dynamic Island for authenticity
- Keep text overlay minimal — the app UI should speak for itself
