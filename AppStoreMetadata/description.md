## Description (4000 chars max)

InterfaceForge is the native iOS app for turning plain-language prompts into polished, production-ready UI components.

DESCRIBE IT. GENERATE IT. EXPORT CODE.

Type what you need — "a pricing card for a developer tools SaaS with 3 tiers" — and InterfaceForge generates a complete, responsive UI component. Preview it live, refine it with follow-up prompts, then export starter code in your framework of choice.

FOUR EXPORT FORMATS
• React + CSS — Component with className-based styles
• Tailwind CSS — Utility-first responsive markup
• HTML + CSS — Zero-dependency vanilla web code
• SwiftUI — Native Apple UI framework code

Every export includes design tokens, a README with setup instructions, and a LICENSE file. Drop it into your project and start building.

AI-POWERED GENERATION
Connect any OpenAI-compatible API (OpenAI, local models, custom endpoints) with your own key, or upgrade to Pro for hosted AI with no setup required. When AI is unavailable, InterfaceForge generates clearly labeled template fallbacks — never fake AI output.

ITERATIVE REFINEMENT
"Make the header darker." "Add a testimonial section." "Change to a 2-column layout." Follow-up prompts let you refine your component without starting over. Multi-turn conversation keeps full context.

LIVE WEB PREVIEW
See your generated component rendered in a real web view. Toggle between mobile, tablet, and desktop viewports to check responsive behavior before exporting.

SCREENSHOT → COMPONENT
Snap a photo of any UI — a website, an app screen, a whiteboard sketch — or pick from your photo library. GPT-4o vision analyzes the layout and auto-fills your prompt. Reverse-engineer any design you admire.

20 STARTER TEMPLATES
Pricing cards, dashboards, onboarding heroes, checkout forms, portfolios, login screens, testimonial carousels, FAQ accordions, comparison tables, email signup forms, and more. Every template is fully customizable.

COMMUNITY GALLERY
Browse, fork, and remix designs shared by other InterfaceForge users. Every shared component includes the original prompt so you can learn, iterate, and build on great ideas.

CLOUDKIT SYNC
Your design history and prompt library sync across all your Apple devices via your private iCloud container. Pick up on iPad where you left off on iPhone.

PRIVACY FIRST
• Your API key is stored in the iOS Keychain — never in plain text
• Prompts are sent only to your configured AI provider
• Design history syncs via your private iCloud — we never see it
• No analytics tracking, no ads, no data collection

FREE TO START
• 3 AI generations per day with your own API key
• All 20 templates and all 4 export formats
• Local design history and prompt library

PRO — $6.99/month
• Unlimited AI generations
• Hosted AI — no API key needed
• Iterative refinement
• Community gallery access
• CloudKit sync across devices
• Priority model access

Requires iOS 17.0 or later. Optimized for iPhone and iPad with a dedicated split-view layout.
