# Release Notes

## Version 1.0.0 — Initial Release

InterfaceForge is here! Describe any UI component in plain English and get production-ready starter code.

What's included:
• AI-powered UI generation from text prompts
• 4 export formats: React + CSS, Tailwind, HTML + CSS, SwiftUI
• 20 starter templates across common component types
• Live web preview with responsive viewport controls
• Iterative refinement with follow-up prompts
• Screenshot-to-component via GPT-4o vision
• Community gallery — browse, fork, and remix shared designs
• CloudKit sync for design history and prompt library
• iPad split-view layout with sidebar navigation
• Privacy-first: Keychain storage, no tracking, no ads

Free: 3 generations/day with your own API key.
Pro ($6.99/mo): Unlimited generations, hosted AI, no key needed.
